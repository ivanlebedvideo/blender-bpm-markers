# BPM Timeline Markers

Blender addon for automatic BPM-based timeline marker placement.

## Description

This addon automatically places timeline markers in Blender based on the BPM (Beats Per Minute) of your music track. Perfect for creating music visualizations and animations synchronized to music beats.

## Features

- Set BPM of your music track
- Configure step size (every beat, every 2 beats, half beats, etc.)
- Automatic marker placement on timeline
- Available in both 3D Viewport and Timeline editor
- Clear existing markers option

## Installation

1. Download `bpm_markers.py`
2. In Blender, go to Edit > Preferences > Add-ons
3. Click "Install..." and select the downloaded file
4. Enable the addon by checking the checkbox

## Usage

1. Press `N` in 3D Viewport or Timeline editor to open the sidebar
2. Navigate to the "BPM" tab
3. Set your track's BPM
4. Configure the step (1.0 = every beat, 2.0 = every other beat, 0.5 = twice per beat)
5. Choose whether to clear existing markers
6. Click "Create markers"
